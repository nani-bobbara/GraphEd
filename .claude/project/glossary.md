# GraphEd Glossary

- Topic: A learning subject created by a user.
- Module: A sub-unit of a topic with specific learning objectives.
- Video: A YouTube video referenced by metadata only.
- Mapping: The association between a module and one or more videos.
- Mindmap: A graph representation of modules and relationships.
- Status: The lifecycle state of a topic (pending_syllabus, syllabus_ready, etc.).
