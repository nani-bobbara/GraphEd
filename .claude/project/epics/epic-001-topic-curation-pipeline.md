# Epic 001: Topic Curation Pipeline

## Goal
Transform a user-entered topic into a curated, verified, structured learning path.

## Included Features
- Feature 001: Topic Creation
- Feature 002: Syllabus Generation
- Feature 003: YouTube Search and Ingestion
- Feature 004: Video Verification and Ranking

## Acceptance Criteria
- User can create a topic and see it progress through all phases.
- All AI outputs are validated before persistence.
- All data is stored in Supabase with correct relations.
