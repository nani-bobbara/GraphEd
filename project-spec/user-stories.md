# GraphEd — User Stories

## Personas
- **Student**
- **Instructor**
- **Admin**
- **Developer**
- **Agent Orchestrator**

---

## Student Stories
- As a student, I want to take assessments so I can measure my understanding.
- As a student, I want personalized feedback so I can improve quickly.
- As a student, I want explanations for each question.

## Instructor Stories
- As an instructor, I want to generate assessments automatically.
- As an instructor, I want to review and edit AI‑generated content.
- As an instructor, I want analytics on student performance.

## Admin Stories
- Manage users, roles, and permissions.
- Configure platform‑wide settings.

## Developer Stories
- As a developer, I want automated deployments.
- As a developer, I want schema migrations handled safely.

## Agent Orchestrator Stories
- Validate MCP readiness before any task.
- Assign tasks to the correct agent based on skillset.
- Prevent unsafe or unauthorized operations.
