# GraphEd — Platform Overview

GraphEd is an AI‑native, agent‑orchestrated learning and assessment platform designed to generate, evaluate, and personalize educational content using multi‑agent reasoning, Supabase backend services, and Next.js frontend delivery.

## Core Objectives
- Provide dynamic, personalized learning experiences.
- Automate content generation (questions, explanations, assessments).
- Enable real‑time evaluation and feedback loops.
- Integrate seamlessly with modern DevOps (GitHub Actions, Vercel).
- Support agentic workflows using MCP servers for safe execution.

## Primary Components
- **Next.js App Router frontend**
- **Supabase backend (DB, Auth, Storage, Edge Functions)**
- **Agent Orchestrator**
- **Domain‑specific agents (Content, Evaluation, DevOps, Infra, Finance)**
- **MCP integrations (GitHub, Vercel, Supabase, Stripe)**

## Guiding Principles
- AI‑first architecture
- Modular, agent‑driven workflows
- Secure, permissioned operations via MCP
- Developer-friendly extensibility
