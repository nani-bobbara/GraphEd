# GraphEd — Agent Roles

## Orchestrator Agent
- Validates MCP readiness
- Assigns tasks to domain agents
- Ensures safety boundaries

## Content Agent
- Generates questions, explanations, hints
- Tags content with metadata

## Evaluation Agent
- Grades responses
- Generates feedback
- Adjusts difficulty

## DevOps Agent
- Manages GitHub Actions
- Handles CI/CD
- Coordinates Vercel deployments

## Infra Agent
- Manages Supabase migrations
- Updates schemas
- Deploys edge functions

## Finance Agent
- Manages Stripe operations
- Creates products
- Handles test payments
