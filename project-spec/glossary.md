# GraphEd — Glossary

**Agent Orchestrator**  
Central controller that assigns tasks and validates MCP readiness.

**MCP (Model Context Protocol)**  
A secure interface for agents to interact with external systems.

**Supabase**  
Backend-as-a-service providing Postgres, Auth, Storage, and Edge Functions.

**Vercel**  
Deployment platform for Next.js applications.

**GitHub Actions**  
CI/CD automation system.

**Stripe**  
Billing and subscription management platform.
